🚘 DriveGuard AI
Agentic Automotive Predictive Maintenance & Proactive Service Scheduling Platform

DriveGuard AI is an advanced automotive intelligence platform designed to predict vehicle faults before they occur, automate service scheduling, assist drivers via voice interactions, and provide manufacturing feedback for continuous product improvement.

🔍 Problem Statement

Vehicle breakdowns, last-minute servicing, and slow diagnostics lead to:

High maintenance costs

Poor customer satisfaction

Inefficient service center operations

Lack of feedback for OEM design improvements

Traditional methods focus on reactive maintenance — DriveGuard AI transforms it into predictive + proactive maintenance.

🧠 Solution Overview

DriveGuard AI uses a multi-agent AI system to:

Detect anomalies and possible faults in real time

Analyze severity and risk levels

Automatically schedule service appointments based on urgency

Forecast spare-part demand for supply chain optimization

Provide voice assistance and alerts to drivers

Give actionable analytics to manufacturers

🧩 Key Features
Feature	Description
🔍 Diagnostics Agent	Detects early-stage component issues
⚠ Severity Agent	Risks scoring and safety threat analysis
🧠 Root Cause Agent	Suggests probable failure causes
🛠 Service Scheduler Agent	Optimizes service slots & routing
📈 Demand Forecast Agent	Predicts spare-part requirements
🎙 VoiceBot Agent	Conversational troubleshooting

🧪 How to Run Locally

cd DriveGuard-AI

Backend Setup
cd backend
python -m venv venv
venv\Scripts\activate   # Windows
pip install -r requirements.txt
uvicorn app.main:app --reload

Frontend Setup
cd frontend
npm install
npm start
